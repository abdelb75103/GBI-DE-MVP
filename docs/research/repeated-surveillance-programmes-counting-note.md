# Repeated Surveillance Programmes: Counting Rule for UEFA ECIS, NCAA, HS RIO, and Similar Study Families

Prepared: 2026-04-13

## Purpose

This note captures a defensible counting rule for repeated-surveillance paper families such as:

- UEFA ECIS / WECIS
- NCAA Injury Surveillance Program papers
- High School RIO papers
- FIFA tournament surveillance papers

The goal is to avoid letting prolific surveillance systems dominate descriptive evidence-map metrics such as:

- population
- injury definition
- incidence definition
- severity definition
- mechanism-reporting frame

## Recommended primary rule

Use `unique surveillance programme-period` as the main counting unit, not `paper` and not automatically `year`.

In practice:

- Count `1` when multiple papers come from the same surveillance programme, cover the same or substantially overlapping surveillance period, and use the same population/methodological frame.
- Split into a new count only when there is a real analytical change:
  - non-overlapping surveillance period that should stand as a distinct observation block
  - different population frame
  - different sex or age stratum
  - different competition frame
  - different injury/illness definition
  - different denominator or exposure framework

## Why this is the best-defended approach

### 1. General evidence-synthesis guidance is clear on overlap

Peer-reviewed methods literature consistently warns that overlapping reports from the same underlying data can overstate the evidence and distort counts if treated as independent observations.

### 2. Sports injury epidemiology often publishes multiple papers from the same surveillance system

UEFA ECIS, NCAA-ISP, HS RIO, and FIFA tournament surveillance are ongoing programmes that generate many papers from the same infrastructure, often on overlapping or contiguous surveillance periods. Counting each paper as a separate methodological observation would overweight those systems relative to one-off cohorts.

### 3. Year-level counting is useful only for time-trend questions

If the question is specifically about how surveillance practice changed over time, then surveillance-year or programme-period-by-year counting can be justified as a sensitivity analysis. It is not the cleanest default for broad descriptive evidence maps.

## Working interpretation for this project

For broad methodology tables and slides:

- `do not count one paper = one count`
- `do not default to one year = one count`
- `count one unique surveillance programme-period = one count`

Use additional linked papers only to:

- fill missing fields
- verify wording
- recover subgroup details
- check whether methods changed enough to justify a split

## Inference boundary

I did **not** locate a sport-specific paper that literally says: "count repeated surveillance publications once per programme-period."

So the recommendation here is an **inference**, built from two evidence streams:

1. peer-reviewed methodological guidance against double-counting overlapping data
2. peer-reviewed sports surveillance papers showing that these paper families are outputs from ongoing surveillance programmes rather than independent stand-alone cohorts

That inference is methodologically defensible and, in my view, stronger than either `one paper = one count` or `one year = one count` as a default evidence-mapping rule.

## Suggested decision rule

### Merge into one count when all are true

- same surveillance programme family
- same or substantially overlapping surveillance period
- same competition/population frame
- same methodological frame
- later papers are topic-specific or secondary analyses of the same surveillance stream

### Split into separate counts when any are true

- clearly non-overlapping programme-periods are being represented as separate surveillance blocks
- the surveillance definition changed materially
- the denominator/exposure approach changed materially
- the population frame changed materially
- the competition type changed materially
- men and women, youth and senior, or other core strata are genuinely separate surveillance cohorts

## Practical examples

### UEFA ECIS

- Multiple ECIS papers based on the same long-running men's elite-club surveillance infrastructure should usually be treated as one programme family, then split only when the relevant programme-period or population/method changes.
- Topic papers on MCL injury, return-to-play time, hamstrings, groin injuries, etc. should not each become separate counts for high-level methodology metrics if they derive from the same ECIS surveillance stream.

### NCAA / HS RIO

- Sport-specific papers from NCAA-ISP or HS RIO should usually be linked back to the same surveillance programme-period instead of counted independently for broad methodology tables.
- Split only when the paper clearly represents a distinct, non-overlapping period or a materially different surveillance frame.

### FIFA tournaments

- Tournament surveillance is best conceptualised as repeated surveillance periods within one broader programme.
- A pooled 1998-2012 surveillance summary and its component tournament papers should not all be counted independently for the same methodological descriptor.

## Slide-ready wording

We treated repeated outputs from the same surveillance systems as a single `surveillance programme-period` unless there was a clear change in the surveillance period, population, or methodological frame. This avoided double-counting overlapping data and prevented prolific programmes such as UEFA ECIS, NCAA-ISP, HS RIO, and FIFA tournament surveillance from disproportionately shaping descriptive evidence-map metrics.

### A. Peer-reviewed methodological guidance on overlap and duplicate counting

1. Senn SJ. *Overstating the evidence: double counting in meta-analysis and related problems*. BMC Medical Research Methodology. 2009;9:10.
   - Core use here: direct methodological support that double-counting overlapping evidence inflates the apparent evidence base.
   - URL: https://pubmed.ncbi.nlm.nih.gov/19216779/
   - PMCID: https://pmc.ncbi.nlm.nih.gov/articles/PMC2653069/

2. Lunny C, Pieper D, Thabet P, et al. *Managing overlap of primary study results across systematic reviews: practical considerations for authors of overviews of reviews*. BMC Medical Research Methodology. 2021;21:140.
   - Core use here: overlap should be identified, visualised, explained, and actively managed; there is no single universal method, so a transparent pre-specified rule is appropriate.
   - URL: https://pubmed.ncbi.nlm.nih.gov/34233615/
   - Full text: https://pmc.ncbi.nlm.nih.gov/articles/PMC8265144/

3. Murphy L, Wyllie A. *Duplicate patient data in a meta-analysis; a threat to validity*. Journal of Critical Care. 2009;24(3):466-467.
   - Core use here: short direct warning that duplicate patient data threatens validity.
   - URL: https://pubmed.ncbi.nlm.nih.gov/19327334/

## How to cite this decision in methods text

Possible wording:

> To avoid double-counting overlapping evidence, repeated publications from the same surveillance systems were grouped at the level of the surveillance programme-period rather than treated as independent papers. Additional publications from the same programme were used to supplement missing methodological details or outcome-specific information, and separate counts were created only when the surveillance period, population, or methodological frame changed materially.

## If we want a sensitivity analysis later

Two optional checks would be reasonable:

- `primary`: programme-period counting
- `sensitivity`: surveillance-year weighted counting for time-trend-oriented descriptors

If both approaches point in the same direction, the slide argument becomes stronger.
