# Review Backlog

This file is the canonical review tracker. It tracks papers extracted, tagged, or staged by Codex that still need human review.

Review state:
- `⏲️ pending_review`: staged, extracted, or tagged by Codex and still needs manual check
- `✅ reviewed_complete`: you reviewed it and it is done
- `needs_follow_up`: reviewed, but something still needs changing

How we use this:
- Codex adds new papers in batches.
- You tell me when a paper is reviewed or complete.
- I update this file so we keep a running backlog.
- Batch-specific review files are optional working notes; the backlog is the source of truth.
