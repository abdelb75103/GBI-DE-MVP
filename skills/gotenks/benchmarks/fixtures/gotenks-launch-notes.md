# Gotenks Launch Notes

- Target users: operators, PMs, founders, and marketers who turn notes into polished markdown deliverables.
- Core value proposition: combine Codex structure with Claude prose quality in one workflow.
- v1 scope: prompt, local text sources, optional shared web research, final markdown output, benchmark harness.
- Out of scope: code generation, binary document parsing, dashboards, and scheduled automation.
- Launch risks: unclear expectations around cost, confusion about when dual-model synthesis is worth it, and benchmark setup time.
- Desired launch message: "Use one skill when the writing quality matters enough to justify a second pass."
