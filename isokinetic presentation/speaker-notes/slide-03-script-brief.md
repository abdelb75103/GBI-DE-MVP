# Slide 3 Script Brief

## Slide purpose

This is the third slide in the Isokinetic 2026 deck.

It is the mismatch / problem-framing slide in Group A:

- football is global
- the evidence base is not yet globally integrated

The script should support staged reveals tied to speech cues.
The slide itself should stay sparse: one paper-based surveillance message first, then one stronger literature-concentration message.

## Core message

Football is global, but its evidence base is not yet globally integrated.

## Surveillance anchor

Use this framing line or a very close equivalent:

> Global surveillance activity exists, but it remains concentrated in a limited number of countries, uneven across confederations, and incomplete in what it captures.

Use these surveillance-paper results as fixed inputs:

- 53 countries
- all 6 confederations represented
- 49 ongoing surveillance programmes
- those 49 ongoing programmes were concentrated in 20 countries

## Transition line

Use this bridge line or a very close equivalent:

> But the prospective literature is even more concentrated than the surveillance landscape.

- Show the surveillance paper with one compact data point:
