# First Gotenks Prompt

Write a polished planning brief for an academic conference talk on the FIFA Global Burden of Injury and Illness in Football Living Systematic Review.

Constraints:

- Focus only on Phase 1, the Living Systematic Review
- Exclude Phase 2 and Phase 3 from the talk body
- Use the local project documents as the source of truth for the project scope
- Use the 2026 paper `Player health surveillance in football: a global map of programmes in national and international leagues and tournaments` as a key framing reference
- Use the current export only as a provisional indicator of what descriptive patterns are available, not as final analysis truth
- Keep the talk academic, professional, clean, and memorable
- The dashboard should be treated as one dissemination output of the living review, not the main subject of the talk

The brief should decide:

- the strongest narrative for the talk
- how to introduce FIFA GBI and what makes it novel
- how to use provisional findings without overclaiming
- how to balance project introduction, preliminary results, and the consensus/comparability story
- what kinds of visuals and Remotion-friendly animation beats fit the talk
- what practical takeaway the audience should leave with

Do not output a final script yet. Output the master planning brief that the later talk build will follow.
